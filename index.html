Aspecto Visual

Creamos imagenes bonitas y delicadas usando una camara Canon t3, este es apenas el primer paso.

Edición Fotografica

Usamos programas de edición para crear distintas versiones de cada foto, vigilamos desde los colores hasta el mas minimo detalles de la imagen.

Dropbox

Usamos el servicio de almacenamiento de dropbox para entragar imagenes. Es tan sencillo como abrir un enlace, para tener acceso a tus fotos.
